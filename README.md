Project Overview: Redco Developments Network Design

This project focuses on designing and implementing a comprehensive network for Redco Developments, a Sri Lanka-based mobile app development company. The main goal is to restructure the existing network at the Kandy head office and design a new network for the Galle branch, ensuring seamless connectivity, security, and support for modern smart technologies.

Key objectives include:

Providing reliable inter-branch connectivity between Kandy and Galle.

Designing department-wise VLANs for efficient network management and security.

Implementing Wi-Fi access for employees and customer service areas.

Supporting smart devices such as automated lighting, smart gates, IP cameras, and ERP software integration.

Ensuring scalability and future-proofing for company growth in the Southern Province.

The network design considers:

Department-specific requirements and employee counts

Server room setup for centralized resources

Video conferencing and smart office solutions

Security measures including IP cameras and secure Wi-Fi

The project is implemented using Cisco Packet Tracer, simulating realistic network devices, connectivity, and configurations to demonstrate a fully functional enterprise network environment.
